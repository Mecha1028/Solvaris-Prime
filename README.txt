Solvaris Prime is a top down horde survival shooter. Enemies spawn periodically in random points of the map, enemies will move towards the player attempting to collide with the player, dealing damage. Enemies have a chance to drop a power up which provide one of three permanent buffs. The objective is to live as long as possible.

How to Run Game
1. Unzip file
2. Open file "GameBuild"
3. Run Solvaris Prime.exe


Controls:
WASD - Move
Mouse - Aim
LMB - Shoot
